# eSkool Tools

## Roles
 
- Admin
- Closer
- Setter

## Permissions

- Admin
    - All Permissions but with some extra
        - Change comission
        - Remove closer/ setter or disable account
- Closer
    - Claim order
    - View order
    - Commission breakdown
    - Comission export
- Setter
    - Same permissions as closer

```bash
make fix
```
